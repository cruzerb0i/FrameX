# Frame X

Frame X is an e-commerce web application specifically designed for the Egyptian market, featuring a mobile-first UI for posters, frames, and stickers. It also includes a custom product builder.

## Features
- Modern, dynamic design with high conversion UI/UX
- Bilingual support context capabilities
- Local Egyptian payment method integration (Vodafone Cash, Instapay, COD)
- Custom picture framing tool

## How to use
This is a standard Vanilla HTML/CSS/JS web project without external build dependencies.
Simply serve the `index.html` file using any web server. For instance:
- `npx serve .` 
- Or use VS Code's "Live Server" extension.
